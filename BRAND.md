# 岳麓食纪品牌与视觉基准

文档状态：品牌基准 v0.2  
最后更新：2026-04-25  
适用范围：前端界面、PRD、设计稿、后续 UI 重构

## 1. 品牌定位

岳麓食纪不是外卖平台，也不是商家营销平台。它的核心心智是“本地大学生真实吃过的美食地图”。

品牌关键词：

- 真实
- 本地
- 学生可信
- 地图发现
- 轻社交
- 日常高频

视觉上应更接近“校园地图 + 真实记录 + 本地生活工具”，避免走美团式高饱和黄橙促销感。

## 2. 主旨色

主色定为：岳麓青绿。

## 3. 与美团色系的边界

需要避免：

- 以黄色或橙色作为品牌主色。
- 大面积 `orange-500`、`yellow-500` 渐变。
- 促销式黄色按钮作为主 CTA。
- 橙黄渐变背景作为首屏或全局 loading 背景。

允许保留：

- 评分星标使用 amber。
- 收藏状态使用 amber。
- 个别食物热度、提醒类标签使用 amber，但不能成为页面主导色。

## 4. 界面风格

推荐风格：清爽地图工具 + 校园记录感。

执行规则：

- 地图是主画布，浮层和侧栏应克制，不抢地图。
- 卡片圆角控制在 `8px-12px`，大浮层可用 `16px`。
- 阴影使用两档即可：卡片轻阴影、浮层中等阴影，避免全局 `shadow-2xl` 泛滥。
- 动效控制在 `150ms-300ms`，只用于状态切换、浮层进出、按钮反馈。
- 演示数据统一使用中性或蓝色标签，不使用主品牌色伪装成真实可信数据。

## 5. 后续落地规则

落地规则：

- 在 Tailwind 配置或 CSS variables 中定义 `brand.*` 语义色。
- 将主链路中的 `orange-*` 替换为 `teal-*` 或 brand token。
- 将收藏/评分中的 `yellow-*` 收敛为 `amber-*` 语义色。
- 禁止在新组件中继续硬编码品牌色；优先使用 token 或统一 Tailwind 映射。

当前状态：

- `brand.*` token 已建立。
- 主链路已切换到岳麓青绿。
- 黄色保留为评分、收藏、提醒语义。
